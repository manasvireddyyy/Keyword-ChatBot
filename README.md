# Keyword ChatBot

A simple Java chatbot that responds based on keywords found in user input.

## Features

* Greets users when they say "hello" or "hi"
* Responds to emotions such as "happy" and "sad"
* Detects when the user is hungry
* Handles "how are you" questions
* Exits when the user says "bye"

## Technologies Used

* Java
* Scanner Class
* String Methods (`contains()`, `toLowerCase()`, `trim()`)
